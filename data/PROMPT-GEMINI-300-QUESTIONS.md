# برومبت Gemini — إنشاء 300 سؤال للداتابيس

انسخ النص أدناه كاملاً والصقه في Gemini. سيُرجع لك JSON جاهز للصق في `data/questions_add_500_new.json`.

---

## البرومبت (انُسخ كل ما بين الخطين)

```
أنت خبير في إنشاء أسئلة مسابقات ثقافية إسلامية وتعليمية. المطلوب: إنشاء 300 سؤال باللغة العربية بصيغة JSON محددة.

# المطلوب بالضبط

1. **المخرجات:** مصفوفة JSON واحدة فقط، بدون أي نص قبلها أو بعدها. يبدأ الملف بـ `[` وينتهي بـ `]`.

2. **نوعان من الأسئلة:**

   أ) **صح/خطأ (tf):** كل سؤال يحتوي:
   - `type`: `"tf"`
   - `category`: أحد الـ 13 تصنيفاً التالية فقط: القرآن الكريم، السيرة، الفقه، الحديث الشريف، الصحابة، العقيدة، التاريخ الإسلامي، الثقافة العامة، الجغرافيا، اللغة العربية، العلوم، الرياضيات، معلومات إسلامية
   - `difficulty`: أحد: `"easy"` أو `"medium"` أو `"hard"` أو `"extreme"`
   - `question_ar`: نص السؤال بالعربية
   - `correctBoolean`: `true` أو `false` (قيمة منطقية، بدون علامات اقتباس)
   - `source_ar`: اختياري (مثال: "مسابقات أهل القرآن" أو اتركه فارغاً أو احذفه)

   ب) **اختيار من متعدد (mcq):** كل سؤال يحتوي:
   - `type`: `"mcq"`
   - `category`: (نفس الـ 13 تصنيفاً أعلاه)
   - `difficulty`: `"easy"` أو `"medium"` أو `"hard"` أو `"extreme"`
   - `question_ar`: نص السؤال بالعربية
   - `options_ar`: مصفوفة من 4 خيارات بالعربية بالضبط — مثال: `["خيار أ", "خيار ب", "خيار ج", "خيار د"]`
   - `correctIndex`: رقم من 0 إلى 3 يحدد ترتيب الإجابة الصحيحة في options_ar
   - `source_ar`: اختياري

3. **التقسيم المطلوب:**
   - 150 سؤال من نوع `tf` (صح/خطأ)
   - 150 سؤال من نوع `mcq` (اختيار من متعدد)
   - توزيع معقول على الـ 13 تصنيفاً
   - توزيع معقول على المستويات: easy / medium / hard / extreme

4. **قواعد صارمة:**
   - لا تضف حقل `id` — سيُضاف تلقائياً لاحقاً
   - في MCQ: يجب أن يكون هناك 4 خيارات دائماً، ولا تتكرر نفس الجملة
   - `correctIndex` في MCQ = 0 أو 1 أو 2 أو 3 (فهرس الإجابة الصحيحة)
   - اجعل الأسئلة واضحة وقابلة للإجابة من طالب ثقافي
   - تجنب الأسئلة المبهمة أو الطويلة جداً

5. **أمثلة:**

   صح/خطأ:
   {"type":"tf","category":"السيرة","difficulty":"easy","question_ar":"ولد النبي محمد صلى الله عليه وسلم في مكة المكرمة.","correctBoolean":true,"source_ar":"مسابقات أهل القرآن"}

   اختيار من متعدد:
   {"type":"mcq","category":"القرآن الكريم","difficulty":"medium","question_ar":"كم عدد سور جزء عم؟","options_ar":["37","36","35","38"],"correctIndex":0,"source_ar":"مسابقات أهل القرآن"}

اطبع المصفوفة JSON كاملة (بدون markdown، بدون ```)، بحيث يمكن نسخها مباشرة إلى ملف JSON.
```

---

## بعد الحصول على النتيجة

1. انسخ ناتج Gemini (المصفوفة JSON).
2. افتح الملف `data/questions_add_500_new.json` (أو أنشئه إن لم يكن موجوداً).
3. احذف محتواه والصق الناتج بداخله.
4. تأكد أن الملف يبدأ بـ `[` وينتهي بـ `]` ولا يوجد نص إضافي.
5. شغّل الفحص أولاً:
   ```
   node tests/merge-questions.js data/questions_add_500_new.json
   ```
6. إذا ظهر كل شيء سليم، طبّق الدمج:
   ```
   node tests/merge-questions.js data/questions_add_500_new.json --apply
   ```
